# 🏛️ Arquitetura Lakehouse

<br><br>

## Visão Geral

A arquitetura Lakehouse combina o melhor dos dois mundos:

- Data Lake  
- Data Warehouse  

Essa abordagem permite uma plataforma unificada para armazenamento, processamento e análise de dados.

<br><br>

## Benefícios

- flexibilidade para lidar com múltiplos tipos de dados  
- capacidades modernas de análise  
- suporte a machine learning  
- alta escalabilidade para grandes volumes de dados  

<br><br>

## Aplicação neste Projeto

A solução é construída utilizando:

- MinIO (camada de armazenamento Data Lake)  
- Spark (motor de processamento distribuído)  
- Parquet (formato colunar otimizado)  
- pipelines de análise distribuída  

<br><br>

##  Conceitos de Armazenamento de Dados

- **Data Lake** → armazenamento centralizado de dados brutos em qualquer formato (estruturado, semiestruturado ou não estruturado), otimizado para escalabilidade e flexibilidade.

- **Data Warehouse** → sistema de armazenamento de dados estruturados, organizados e tratados, otimizado para consultas rápidas, relatórios e inteligência de negócios.
